Limit Buy

A Limit Buy order lets you set a buy price below the current market price. When the token drops to your target, SniperFi automatically buys it for you.

Use Case

You're watching $DOGE420 at 0.0005 SOL. You think it'll dip to 0.0003 SOL before pumping. Set a Limit Buy at 0.0003 SOL with 1 SOL amount. If it dips â€” you buy. If it doesn't â€” nothing happens.

Execution

When a target price is hit, the execution flow is:

Flow 1. Price monitor detects target match 2. Jupiter quote requested for best route 3. Transaction built with user's token account 4. Transaction sent to Solana network 5. Confirmation received â†’ order marked as filled 6. User notified (dashboard updates)

Execution speed depends on Solana network conditions. Under normal conditions, orders execute within 1-3 seconds of price target being hit.

âš 

During high congestion, transactions may take longer or fail. Failed transactions are retried automatically up to 3 times.
